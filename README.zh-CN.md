随着时间推移，在**AI编程**，或者称为 `AI Coding`方面出现了越来越多、越来越好用的产品。比如`Cursor`等等。为了先搞明白都是有哪些工具、扩展、或者原生编辑器、云编辑器、终端……。我们开发一个用于分类展示主流AI编程工具的现代化Web应用。

## 功能特点

- 🎯 精选主流AI编程工具合集
- 🌈 氛围编码（Vibe Coding）工具：创造完美编程氛围和心流状态
- 🧠 沉浸式编程环境：发现能提升编程体验的工具
- 💻 开发者体验：专注于提高编码效率和乐趣的工具
- 🔍 按使用方式分类（浏览器/编辑器/扩展/终端）
- 🏢 按公司分类浏览工具
- 🔄 按输出类型分类查看工具
- 💫 现代化的UI设计和流畅的用户体验
- 🌓 支持深色模式
- ⚡ 基于Next.js构建，性能优异

## 快速开始

### 环境要求

- Node.js 18.0.0 或更高版本
- pnpm 8.0.0 或更高版本

### 安装

```bash
# 克隆项目
git clone https://github.com/yayxs/ai-coding.git

# 进入项目目录
cd ai-coding

# 安装依赖
pnpm install
```

### 开发

```bash
# 启动开发服务器
pnpm dev
```

访问 http://localhost:3000 查看应用。

### 构建

```bash
# 构建生产版本
pnpm build

# 启动生产服务器
pnpm start
```
