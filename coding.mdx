## 背景

随着AI编程（或AI编码）的发展，越来越多的产品可用，例如`Cursor`和其他产品。为了更好地了解存在哪些工具、扩展、原生编辑器、云编辑器、
终端。我陆陆续续写了这篇内容。

当前，社区中的大多数`AI Coding`或`Vibe Coding`一般是导航站点。我最初也想写一个导航站点，但后来觉得导航站点本身不会展示价值。与它们不同的是：我想有一些自己"固执己见的"内容。

## 现状

聊一聊现状，目前`LLM` + `MCP` + `Agent` 是主流。我们已经不再关注技术的细节。从深度慢慢的从广度转换。如果你在开发的过程中，使用一些开源的框架或者是库，我们不再纠结
具体的实现原理，曾经我也在纠结是使用 `Next.js` 还是 `Astro`，但是现在都已经不重要了。我们可以使用 `GitHub MCP` 以及 `Git MCP`阅读一些开源的代码。当然随着`DeepWiki`
的出现，外加`Context7 MCP`。作为程序员，我们已经不用纠结什么技术，甚至什么原理了。

## 原生编辑器

原生编辑器：指的是类似`VSCode`这种，能够安装在用户的电脑上使用。

### Cursor

Cursor 是一款与 ChatGPT 同期推出的产品，随着 Anthropic 的 Claude 3.5 Sonnet 的发布而更加闪耀！我已经付费使用了它，我相信它目前是宇宙中最强大的 AI 编辑器。

- **公司**：Anysphere
- **类型**：编辑器
- **输出类型**：日常编码
- **网址**：[https://www.cursor.com/](https://www.cursor.com/)

### Windsurf

Windsurf 是由 Codeium 开发的产品，于2024年11月13日发布。我在知乎上看到后立即尝试了它，感觉非常惊艳。同时，它巩固了我对 AI 编码的兴趣。

- **公司**：Codeium
- **类型**：编辑器
- **输出类型**：日常编码
- **网址**：[https://codeium.com/](https://codeium.com/)

### Trae

Trae 同时支持 Mac 和 Windows 系统，是字节跳动开发的编辑器。

- **公司**：ByteDance
- **类型**：编辑器
- **输出类型**：日常编码
- **网址**：[https://www.trae.ai/](https://www.trae.ai/)

### Zed

Zed 是一款面向未来的编辑器，目前尚不支持 Windows 系统。

- **类型**：编辑器
- **网址**：[https://zed.dev/](https://zed.dev/)

### 其他编辑器

还有一些值得关注的 AI 编辑器：

- **PearAI**：结合了 Aider、Supermaven、MemO、Continue 和 Perplexity 的功能 [https://trypear.ai/](https://trypear.ai/)
- **Void**：一个开源的 Cursor 替代品 [https://voideditor.com/](https://voideditor.com/)
- **Aide**：由 CodeStory.ai 开发的开源 AI 原生 IDE [https://aide.dev/](https://aide.dev/)

## 总结

AI 编程工具正在快速发展，不断涌现出新的产品和功能。这些工具覆盖了从代码编辑、自动补全、应用生成到文档创建的各个方面，大大提高了开发效率。随着 LLM 和 Agent 技术的进步，我们可以期待未来会有更多创新的 AI 编程工具出现。
